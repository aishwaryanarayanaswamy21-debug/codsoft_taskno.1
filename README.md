# Task 1 - Rule-Based Chatbot

This folder contains a simple console chatbot built with Python's standard library.

## What It Does

- Greets the user with responses like `hi`, `hello`, and `hey`
- Answers basic questions about its name
- Replies with the current time and date
- Responds to a few fixed topics such as Python and AI
- Exits when the user types `bye`, `exit`, or `quit`

## Files

- `chatbot.py` - main chatbot script
- `requirements.txt` - empty because the project uses only the Python standard library
- `demo_video_link.txt` - placeholder for a demo link

## Run

```bash
python chatbot.py
```

## Example Prompts

- `hi`
- `what is your name`
- `what is the time`
- `tell me about python`
- `bye`
